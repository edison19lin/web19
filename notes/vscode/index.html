VS Code 筆記

紀錄常用快捷鍵、設定與擴充功能。

常用快捷鍵

  • Ctrl + Shift + P：開啟命令面板
  • Ctrl + `：切換內建終端機
  • Ctrl + B：切換側邊欄顯示
  • Alt + ↑/↓：移動整行程式碼

常用設定

{
  "editor.tabSize": 2,
  "editor.wordWrap": "on",
  "files.autoSave": "afterDelay"
}

推薦擴充功能

  • Python
  • Prettier - Code formatter
  • GitLens
  • Markdown All in One
